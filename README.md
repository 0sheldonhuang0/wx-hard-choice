## 简介
这个小程序主要是帮助选择困难症患者的，当然包括我自己。虽然市面上这样的小程序有很多，但是自己做一个还是觉得挺有成就感的，虽然并不知道之后会去用多少次。“选择”是一大世界性难题。吾日三问吾身：今天吃什么？我要选择哪一个呢？现在做些什么事情比较好呢？
